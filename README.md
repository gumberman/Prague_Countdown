# Prague Countdown

A beautiful, animated countdown timer website for tracking days until your Prague adventure!

## Features

- Real-time countdown (days, hours, minutes, seconds)
- Customizable target date
- Local storage to remember your date
- Responsive design for all devices
- Beautiful animations and Prague-themed graphics

## How to Use

1. Open `index.html` in your browser
2. Set your target date using the date picker
3. Click "Update Countdown" to start counting down!
